# Thai Documentation Skill (ทักษะการบันทึกข้อมูลภาษาไทย)

ทักษะนี้มีไว้สำหรับจัดการและบันทึกการอัปเดตของโปรเจกต์เป็นภาษาไทย เพื่อความชัดเจนในการสื่อสารกับผู้ใช้

## กฎการใช้งาน (Guidelines)

1. **อธิบายเป็นภาษาไทยทุกครั้ง**: ทุกครั้งที่มีการอัปเดตหรือเปลี่ยนสถานะของงาน ให้สรุปเป็นภาษาไทยที่เข้าใจง่าย
2. **บันทึกประวัติการอัปเดต**: บันทึกการเปลี่ยนแปลงที่สำคัญลงในไฟล์ `UPDATES.md` ภายในโฟลเดอร์นี้
3. **ความเป็นมืออาชีพ**: ใช้ภาษาที่เป็นกันเองแต่มีความเป็นมืออาชีพ (Casual but Professional) ตามสไตล์ของสถาบัน CAP Vision

## ไฟล์สำคัญ (Key Files)

- `SKILL.md`: คำแนะนำการใช้ทักษะ (ไฟล์นี้)
- `UPDATES.md`: บันทึกประวัติการอัปเดตงานทั้งหมดเป็นภาษาไทย
